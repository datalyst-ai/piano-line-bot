# strain-hoikuen-dx-poc

保育園DX案件向けPoC。  
LINE Bot上で保育園向け補助金案内を行う試作。

## 目的
園の条件や相談内容を入力すると、該当可能性のある補助金候補を整理して返すPoCを作る。

## 現在の対象範囲
- 川崎市
- 認可保育園
- 人材確保・処遇改善関連の補助金
- LINE Bot上での対話型案内

## 使用技術
- Node.js
- LINE Messaging API
- Claude API
- Vercel

## 現在できていること
- LINE上でメッセージを送ると応答が返る
- 補助金に関するそれらしい回答が生成される

## 未実装または未整理
- デモ用シナリオの固定化
- 回答内容の対象範囲の絞り込み
- 不要な一般論や過剰な列挙の抑制
- 制度候補の絞り込み精度向上
- 打ち合わせ後の要件反映
